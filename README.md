# 📝 To-Do List Project

A simple and responsive To-Do List application built using **HTML**, **CSS**, and **JavaScript**.  
This project helps users manage daily tasks efficiently with an interactive user interface.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎨 Clean UI design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

To-Do-List-Project/
